                                                                 实验三

一、实验目的

掌握 SIFT 算法的原理及在图像特征提取中的应用方法。

理解 FLANN 匹配器的工作机制，实现图像特征点的高效匹配与筛选。

学会使用单应矩阵估计图像间的透视变换关系，完成多幅图像的自动拼接。

熟悉 OpenCV 库中图像读取、格式转换、特征处理及可视化相关函数的使用。

二、实验原理

1. SIFT 特征提取
SIFT（尺度不变特征变换）算法能在不同尺度、旋转和光照条件下提取图像的稳定特征点。通过构建高斯差分金字塔检测极值点，确定特征点的位置和尺度，再计算特征点的主方向，最终生成 128 维的特征描述符，实现特征的尺度和旋转不变性。

2. FLANN 特征匹配
FLANN（快速最近邻搜索库）是一种高效的特征匹配算法，相比暴力匹配能显著提升匹配速度。实验中采用 K 近邻匹配（k=2），为每个特征点返回两个最佳匹配结果，再通过 Lowe's 比率测试（阈值 0.7）筛选出优质匹配点，剔除错误匹配。

3. 单应矩阵与透视变换
单应矩阵（Homography Matrix）是描述两张平面图像间投影关系的 3×3 矩阵，可通过 RANSAC 算法鲁棒估计。利用单应矩阵能将一幅图像映射到另一幅图像的坐标系中，实现透视变换，为图像拼接奠定基础。

4. 图像拼接流程
先通过特征提取与匹配获得图像间的对应关系，再估计单应矩阵实现图像配准，最后通过平移变换调整图像位置，将配准后的图像与基准图像融合，得到完整的拼接结果。

三、实验环境

核心库：OpenCV（图像处理）、NumPy（矩阵运算）、Matplotlib（结果可视化）

硬件环境：普通 PC（CPU≥i5，内存≥8GB）

五.实验总结

本次实验成功实现了基于 SIFT 特征匹配的图像拼接流程，从图像预处理、特征提取与匹配，到单应矩阵估计、图像变换与融合，完整掌握了图像拼接的核心技术。通过实验验证了 SIFT 算法在特征提取中的稳定性和 FLANN 匹配器的高效性，理解了单应矩阵在图像配准中的关键作用。同时，也认识到图像拼接效果受特征匹配精度、变换矩阵估计准确性等因素影响，为后续优化图像融合质量、处理复杂场景下的拼接问题提供了实践基础。
